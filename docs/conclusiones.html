Conclusión

  • A pesar de las diferencias en los materiales y métodos de construcción de carreteras en diferentes países, el uso de un conjunto de datos diverso facilita la generalización del modelo, lo que le permite funcionar adecuadamente en Colombia. Esto indica la robustez del modelo en entornos no vistos previamente.
  • La presencia de ruido en las imágenes, como paisajes, sugiere que un se debería realizar un proceso adicional de preprocesamiento para eliminar este tipo de ruido sería beneficioso para mejorar la precisión y consistencia del entrenamiento.
  • El uso de almacenamiento en la nube y la eliminación de imágenes procesadas una vez descargadas asegura que solo se almacenen los datos críticos, manteniendo bajo control los costos de almacenamiento y optimizando el flujo de trabajo del sistema.
